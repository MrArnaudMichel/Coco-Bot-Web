Editions

CocoBot has two editions:

  • CocoBot Plus: This is the less expensive edition of CocoBot. It has all the basic features of CocoBot, but it does not have all the features of CocoBot Premium.

  • CocoBot Premium This is the more expensive edition of CocoBot. It has all the features of CocoBot Plus, but it also has additional features that are not available in CocoBot Plus.
